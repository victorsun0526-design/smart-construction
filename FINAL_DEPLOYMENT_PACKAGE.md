## 📋 部署步骤

### 快速部署 (7分钟)
```
1. 注册GitHub账号 (2分钟)
2. 创建smart-construction仓库 (1分钟)
3. 上传所有文件 (2分钟)
4. 注册Vercel账号 (1分钟)
5. 部署到Vercel (1分钟)
```

### 详细步骤

#### 第1步: 准备GitHub
1. 访问 https://github.com
2. 点击"Sign up"注册
3. 验证邮箱

#### 第2步: 创建仓库
1. 登录GitHub
2. 点击"+" → "New repository"
3. 名称: `smart-construction`
4. 描述: `智慧工地管理平台`
5. 选择"Public"
6. 点击"Create repository"

#### 第3步: 上传文件
**方法A: GitHub Desktop (推荐)**
1. 下载 https://desktop.github.com
2. 安装并登录
3. 克隆仓库
4. 复制所有文件到仓库文件夹
5. 提交并推送

**方法B: 网页上传**
1. 在仓库页面点击"Add file" → "Upload files"
2. 拖拽所有文件
3. 点击"Commit changes"

#### 第4步: 部署到Vercel
1. 访问 https://vercel.com
2. 点击"Continue with GitHub"
3. 授权访问
4. 点击"New Project"
5. 选择你的仓库
6. 点击"Deploy"
7. 等待1-2分钟

## 🛠️ 故障排除

### 常见问题

#### 问题1: 文件上传失败
**解决方案:**
- 使用GitHub Desktop代替网页上传
- 检查文件大小限制
- 分批次上传

#### 问题2: 部署失败
**解决方案:**
- 检查Vercel部署日志
- 确保所有文件路径正确
- 检查`.gitignore`文件

#### 问题3: 网站显示空白
**解决方案:**
1. 按F12打开开发者工具
2. 查看Console标签页的错误信息
3. 检查Network标签页的文件加载
4. 检查Application标签页的LocalStorage

#### 问题4: 登录失败
**解决方案:**
1. 检查浏览器LocalStorage
2. 清除浏览器缓存 (Ctrl+F5)
3. 检查JavaScript控制台错误

### 调试工具
1. **浏览器开发者工具** (F12)
   - Console: JavaScript错误
   - Network: 文件加载
   - Application: LocalStorage

2. **Vercel部署日志**
   - 在Vercel控制台查看
   - 搜索错误信息

3. **GitHub Actions日志**
   - 如果使用GitHub Pages
