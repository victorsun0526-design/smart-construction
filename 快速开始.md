## 🔧 常见问题解决

### 1. 页面显示不正常
- **问题**: 样式错乱、布局异常
- **解决**: 
  1. 按 `Ctrl+F5` 强制刷新
  2. 清除浏览器缓存
  3. 使用Chrome/Firefox最新版本

### 2. 登录失败
- **问题**: 无法登录或跳转失败
- **解决**:
  1. 确认用户名: `admin`，密码: `123456`
  2. 打开浏览器控制台 (F12 → Console)
  3. 查看错误信息
  4. 使用 [debug_login.html](file:///C:/Users/14648/lobsterai/project/智慧工地平台/debug_login.html) 调试

### 3. 图标显示问题
- **问题**: 进度管理图标大小不一致
- **解决**:
  1. 已修复，确保使用最新文件
  2. 使用 [test_icons.html](file:///C:/Users/14648/lobsterai/project/智慧工地平台/test_icons.html) 验证

### 4. 功能不响应
- **问题**: 点击按钮无反应
- **解决**:
  1. 检查JavaScript是否启用
  2. 查看控制台错误
  3. 确认文件路径正确

## 🚨 紧急情况处理

### 无法访问任何页面
1. 检查文件路径是否正确
2. 确认文件是否存在
3. 尝试用其他浏览器打开
4. 检查文件权限

### 所有功能失效
1. 清除浏览器缓存
2. 重启浏览器
3. 检查网络连接
4. 验证JavaScript是否启用

### 数据丢失
1. 当前为前端演示版，数据存储在localStorage
2. 清除缓存会导致数据丢失
3. 正式版需要后端数据库支持
