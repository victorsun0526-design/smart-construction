# 智慧工地管理平台

一个专业的智慧工地平台网站，包含八大核心管理模块：进度、安全、质量、材料、人员、机械设备、拌合站、预制场等管理模块。

## 项目结构

```
智慧工地平台/
├── index.html              # 网站首页
├── login.html              # 登录页面
├── progress.html           # 进度管理模块
├── safety.html            # 安全管理模块
├── quality.html           # 质量管理模块（待创建）
├── material.html          # 材料管理模块（待创建）
├── personnel.html         # 人员管理模块（待创建）
├── equipment.html         # 设备管理模块（待创建）
├── mixing.html           # 拌合站管理模块（待创建）
├── prefab.html           # 预制场管理模块（待创建）
├── css/
│   ├── style.css          # 主样式文件
│   ├── login.css          # 登录页面样式
│   └── module.css         # 模块页面通用样式
├── js/
│   ├── main.js            # 主JavaScript文件
│   ├── login.js           # 登录页面JavaScript
│   ├── module.js          # 模块通用JavaScript
│   ├── progress.js        # 进度管理模块JavaScript
│   └── safety.js          # 安全管理模块JavaScript
└── README.md              # 项目说明文档
```

## 扩展开发

### 添加新模块
1. 创建新的HTML文件（如 `quality.html`）
2. 参考现有模块结构
3. 添加对应的CSS样式
4. 编写JavaScript功能
5. 更新导航菜单

### 样式定制
- 修改 `css/style.css` 中的颜色变量
- 调整 `css/module.css` 中的模块样式
- 添加新的CSS文件并引入

### 功能扩展
- 在 `js/` 目录下创建新的JavaScript文件
- 实现具体的业务逻辑
- 集成第三方API或服务
