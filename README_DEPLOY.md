## 📋 详细部署步骤

### 方案A：GitHub + Vercel（推荐）

#### 1. 注册GitHub（2分钟）
- 访问：https://github.com
- 点击 "Sign up"
- 填写信息注册

#### 2. 创建仓库（1分钟）
- 登录后点击 "+" → "New repository"
- 名称：`smart-construction`
- 点击 "Create repository"

#### 3. 上传文件（2分钟）
**方法1：GitHub Desktop（最简单）**
1. 下载：https://desktop.github.com
2. 登录，克隆仓库
3. 复制所有文件到仓库文件夹
4. 提交并推送

**方法2：网页上传**
1. 在仓库页面点击 "Upload files"
2. 拖拽所有文件
3. 点击 "Commit changes"

#### 4. Vercel部署（1分钟）
1. 访问：https://vercel.com
2. 用GitHub登录
3. 点击 "New Project"
4. 选择仓库，点击 "Deploy"
5. 等待1分钟完成

### 方案B：直接拖拽部署

#### 1. 压缩文件
1. 右键点击 `智慧工地平台` 文件夹
2. 选择 "发送到" → "压缩(zipped)文件夹"
3. 获得 `智慧工地平台.zip`

#### 2. 使用Netlify拖拽部署
1. 访问：https://app.netlify.com/drop
2. 拖拽zip文件到页面
3. 获得免费域名

## 🚨 常见问题

### Q1：部署后页面空白？
A：检查文件是否全部上传，使用相对路径。

### Q2：登录功能异常？
A：清除浏览器缓存，使用Chrome隐身模式测试。

### Q3：移动端显示错乱？
A：确保已添加viewport meta标签。

### Q4：如何更新网站？
A：修改文件后重新提交到GitHub，Vercel会自动部署。
